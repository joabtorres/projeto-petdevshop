# projeto-petdevshop

Projeto feito no curso de Node + Typescript

### Pre-requisitos globais:

`npm install -g typescript nodemon ts-node`

### Instalação

`npm install`

### Para roda o projeto

`npm run start-dev`

## Bibliotécas utilizadas

-  express;
-  mustache-express;
-  dotenv;
-  typescript;

### Types
- @types/node;
- @types/express;
- @types/mustache-express;
